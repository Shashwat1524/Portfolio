DASHBOARD DESIGN

Designed a dashboard utilizing Raspberry Pi and an ESP32 microcontroller. Implemented CAN communication to receive motor performance data, including motor speed and battery current, from the motor controller. The display was coded using Python, and the design was enhanced using the Pygame library for a more interactive and visually appealing dashboard interface.

AQUA-CULTURE COOLER USING DS18B20 temperature sensor and STM32

This system integrates the STM32 microcontroller, DS18B20 temperature sensor, and a relay-managed fan to establish a closed-loop control mechanism. By leveraging the accurate temperature readings from the DS18B20 sensor, the STM32 microcontroller dynamically adjusts the fan, ensuring the water temperature remains within specified parameters.

The Smart Water Temperature Control System stands as a notable advancement in temperature regulation technology, offering a dependable and adaptable solution for maintaining optimal conditions in water-dependent applications.

IIOT WATCH THAT MEASURES PULSE

Developed a smart watch with ESP32 that uses SEN-1157 Pulse Sensor for real-time heart rate monitoring. The heart beat was displayed SSD1306 OLED Display using I2C Communication. WiFi connectivity seamlessly displays the current time and related information.

QUADCOPTER DRONE

I developed a quadcopter by integrating MPU-6050-IMU sensor for stability and control, as well as IR sensor for object detection. Additionally, I implemented a closed-loop control system for the motors using a PID-based algorithm, effectively propelling the drone to achieve precise performance.
